Adding a Macro to Excel

  1. Open Excel and go to the Developer tab in your ribbon. If you do not have that tab go to Adding the Developer tab to learn how to add it.
  2. Open Visual Basic (the top left of the picture above)
  3. Click on "Insert", and "Module"
  4. In "Module1" that appears, paste the code where you see it in the picture
  5. Close "Visual Basic", then open "Macros" (right next to Visual Basic) and select "hype_link"
  6. Click on "Options", then assign a key-bind to it.

    7. Your macro should now be set up and assigned to use with a key-bind! To run it, simply press "Ctrl" and whatever key you decided to assign it to.
