Hyperlinking between two cells

Hyperlink Macro code

Steps for using this macro:

  1. Download the code in the text file above, open it and copy all of the code.
  2. Bind it to a key or button in Excel. For instructions on this go to Adding a Macro to Excel
  3. When you know which cells you want to hyperlink together, run the macro.
  4. An input box will show up asking you for the source cell. Enter the cell (e.g. "c32") that is on the sheet you are currently on.
  5. Next, another input box will ask you for the destination cell. Enter this cell like you did before. (Note: this cell can be on a different sheet)
  6. Finally, an input box will ask you the sheet number of your destination cell. If your destination cell is on the first sheet in the Workboox, enter "1". If it is on the fourth sheet, enter "4", and so on.
